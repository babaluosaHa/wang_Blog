bitMap算法
---------------------------
所谓 BitMap 就是用一个 bit 位来标记某个元素对应的 value，而key即是这个元素。由于采用bit为单位来存储数据，因此在可以大大的节省存储空间。

缺点
---------------------------
无法对存在重复的数据进行排序和查找

int a[N/32 + 1];	// N个数
index_loc = N / 32
bit_loc = N % 32

index_loc = n >> SHIFT; // 等价于n / 32
bit_loc = n & MASK;	// 等价于n % 32 。
